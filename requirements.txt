1. Hafta – Araştırma + Dataset seçimi

Kaggle veya hocanın verdiği datasetlerden uygununu bul.

Veri yapısını incele (hangi kolonlar var, hedef (signal strength) var mı?).

Projeyi yazılı olarak “proposal” haline getir (hoca onayı için).
⏳ 5-6 saat

2. Hafta – Data Cleaning

Eksik değerleri sil/doldur.

Gereksiz kolonları at.

Data’yı train/test setlerine ayır.
⏳ 5-6 saat

3. Hafta – Feature Engineering

Zaman bilgisini (günün saati, hafta içi/sonu) ekle.

Lokasyon verisini (lat/long) kullanılır hale getir.

Network type (3G/4G/5G gibi) encode et.
⏳ 5-6 saat

4. Hafta – İlk Basit Model

Linear Regression veya Decision Tree dene.

Basit hata metriklerini (RMSE, R²) ölç.

Çalıştığını gör.
⏳ 5 saat

5. Hafta – Gelişmiş Model

Random Forest, XGBoost, LightGBM dene.

Hyperparameter tuning yap.

Performansı iyileştir.
⏳ 6-8 saat

6. Hafta – Görselleştirme

Signal strength değerlerini grafiklerle çiz.

Belki heatmap (harita üstünde sinyal yoğunluğu).

Basit dashboard (Matplotlib, Seaborn, Plotly).
⏳ 6 saat

7. Hafta – Rapor + Kod Düzenleme

Kodunu Jupyter Notebook / Python script halinde düzenle.

Proje raporunu yaz (Introduction, Dataset, Methodology, Results, Conclusion).
⏳ 8-10 saat

8. Hafta – Son Hazırlık

Sunum slaytı hazırla.

Demo veya ekran görüntüleri ekle.

Final raporu hocaya teslim et.
⏳ 5 saat
